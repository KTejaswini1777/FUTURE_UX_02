# 📌 Project Overview

Levav Nailart is a modern mobile application designed to simplify the process of booking professional nail art appointments. The application provides users with an elegant and seamless experience, allowing them to browse nail designs, view design details, schedule appointments, and confirm bookings in just a few steps.

The design focuses on creating a premium, user-friendly interface while maintaining a luxurious and feminine visual identity that reflects the Levav Nailart brand.

---

# 🎯 Objective

Design a complete mobile UI/UX experience for a nail art salon booking application that is:

- User-friendly
- Modern and elegant
- Easy to navigate
- Visually appealing
- Consistent in branding

---

# 👥 Target Users

- Women interested in professional nail art
- Bridal and special event customers
- Regular salon clients

- Customers looking for quick online appointment booking
